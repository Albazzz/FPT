     VI: Navigator
  B. Guard ✓

     VI: Route protection
  D. Middleware
     VI: Middleware
Đáp án: B. Guard
【Khái niệm】Route Guard (bảo vệ route) là pattern chặn/cho phép điều hướng dựa trên trạng thái đăng nhập. Khi user chưa login, Guard chuyển hướng sang màn login thay vì mở trang được bảo vệ.

(VI) Mã status nào cho biết lỗi phía server?
  A. 200
     VI: 200
  B. 301
     VI: 301
  C. 404
     VI: 404
  D. 500 ✓
     VI: 500
Đáp án: D. 500
【Khái niệm】HTTP: 2xx success, 3xx redirect, 4xx client error (404 not found), 5xx server error (500 Internal Server Error).

【Vì sao đúng】Server error class → 500 (5xx).
Mẹo: 5xx = lỗi server; 500 điển hình.
  Sai A: 200 OK = thành công.
  Sai B: 301 = redirect vĩnh viễn.
  Sai C: 404 = client: không tìm thấy resource.
